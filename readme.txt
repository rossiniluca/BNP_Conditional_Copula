
Bayesian Nonparametric Conditional Copula Estimation of Twin Data
L. Dalla Valle, F. Leisen and L. Rossini

The data can be downloaded from 

             http://hdl.handle.net/1902.1/13913 

As explained in the webpage, the data must be acknowledged as: 

Loehlin, J. and Nichols, R. (2009). The National Merit twin study. Harvard Dataverse, V3, http://hdl.handle.net/1902.1/13913.

See also the community norms: http://best-practices.dataverse.org/harvard-policies/community-norms.html

The Matlab codes in the paper are in the file

MHrho.m    - computes the M-H for the vector of parameter \beta within the first calibration function
MHrho2nd.m - compute the M-H for the vector of parameter \beta within the second calibration function
condnormcopdens.m - computes the conditional Gaussian/normal density of the copula
Gibbs_Cal.m - Gibbs sampling algorithm
